# R.E.L.I.C

Only available in French. End-of-the-year project for school

Authors: TommyYOyoyo, AFellowhuman, AFellowSealClubber

A lore-based platformer & puzzle style game.

## A meteorite has struck the world, creating a disruption in time and causing people to age faster than they should... The only way to restore peace is to restore the three timecharms scattered into fragments.
